Качество данных и ограничения

Как собрано: плановые показатели извлечены из поля «Описание» задач регулярными выражениями (сумма и вес). Все расчёты ведутся только по Q3; Q4-задел показан справочно и не входит в итоги по менеджерам и в проценты исполнения. Периодные задачи с полем «Месяц:» отнесены к Q3; задачи с «Задел на отгрузку в 4 квартал» — в Q4-задел. Задача [Q3-COM-CORP-GIFTS-TENDERS] учтена по обновлённому описанию: 90 кг икры (август 45 кг · сентябрь 45 кг) ≈ 4 574 880 ₽ без НДС.

Факт: принят равным плановым показателям задач со статусом «Завершена». Комментарии задач пустые; чаты через текущий webhook недоступны (метод im.dialog.messages.list не найден).

Риск: сетевые гипотезы (METRO, МАГНИТ, Гиперглобус) — 51.8% Q3-плана. Канал «Корпоративные подарки и тендеры» — 11.8% Q3-плана. Исполнение этих направлений критично для достижения квартала.
