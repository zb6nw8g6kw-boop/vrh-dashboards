# Операционный дашборд ВРХ · 28 неделя 2026

Статический HTML-дашборд для операционной отчётности Волгореченского рыбного хозяйства.  
Содержит три раздела:

1. **Статус проекта Супергрин** — квартальный план и прогноз Q3 2026.
2. **Статус проекта Грин** — 28 неделя 2026, прогноз Q3 и риски 29 недели.
3. **Операционные показатели 28 недели** — коммерческий блок (квоты, сети, Перекрёсток) — в стадии наполнения.

## Как использовать

### Локальный просмотр

Откройте `index.html` в браузере. Дашборд полностью автономный: никакие внешние CSS/JS-фреймворки не используются, интернет не требуется.

### Публикация через GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите содержимое этой папки (`index.html` и `README.md`) в корень репозитория.
3. Перейдите в **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch → main/master → / (root)**.
5. Через 1–2 минуты дашборд будет доступен по адресу `https://<username>.github.io/<repo>`.

### Тёмная тема

Переключатель темы находится в правом верхнем углу. Выбор сохраняется в `localStorage` браузера.

- `Грин_Коммерция_28_неделя_2026.pdf` — коммерческие показатели 28 недели.

## Обновление

Чтобы обновить дашборд, отредактируйте `index.html` и замените значения в разделах `<div class="tab-panel" id="panel-...">`. Для добавления новых разделов скопируйте структуру существующей вкладки и обновите навигацию `.tabs`.

## Примечание

Раздел «Операционные показатели 28 недели» в текущей версии представлен заглушкой с тизерными KPI. Для наполнения в полном формате требуются исходные данные в формате CSV/XLSX или текстовой таблице.
